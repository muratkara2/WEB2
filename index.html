Amacimiz

Frontendci, tasarimi ve kodu herkes icin anlasilir hale getirir.

Amacimiz, insanlara web site frontend'inin ne oldugunu ogretmek, bunu pratik projelerle pekistirmek ve urettigimiz degerden gelir elde etmektir. Bilgiyi uygulanabilir hale getirip, insanlarin kendi projelerini ayaga kaldirmasini sagliyoruz.

1. Ana başlık

Web Tasarım Nedir?

Web tasarım; bir sitenin görsel dili, içerik hiyerarşisi ve kullanıcıya sunduğu deneyimin planlanmasıdır. İyi tasarım sadece "güzel görünmek" değil, doğru bilgiyi doğru sırada ulaştırmaktır.

2. Ana başlık

Kullanılan Teknolojiler

Frontend projelerinde en temel yapı HTML, CSS ve JavaScript'tir. Bunun üzerine React/Vue gibi kütüphaneler, build araçları ve test sistemleri eklenir.

HTML5 CSS3 JavaScript React Git

3. Ana başlık

Frontend Nasıl Çalışır?

Tarayıcı; HTML'i DOM'a çevirir, CSS ile stil hesaplar, JavaScript ile etkileşimleri çalıştırır. Kullanıcı tıklayınca olay tetiklenir ve arayüz gerçek zamanlı güncellenir.

Adım 1 HTML parse edilir
Adım 2 CSS uygulanır
Adım 3 JS olayları yönetir
